# Lezioni per studenti di liceo scientifico

Questo repository contiene materiale didattico pensato per studenti di **liceo scientifico**, con l’obiettivo di fornire una prima introduzione all’analisi dei dati e al machine learning.

## Contenuti

- Introduzione a **Pandas** per la manipolazione e l’analisi dei dati
- Introduzione al **problema della classificazione**
- Esempi pratici e notebook commentati
